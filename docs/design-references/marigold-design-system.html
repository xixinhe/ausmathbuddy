01 / Color

One canvas, one ink, one confident accent.

The palette is deliberately small: a saturated marigold canvas, an olive-brown ink that does double duty as text and dark-card fill, and a single orange reserved for numbers and the one CTA link.

02 / Typography

Rounded weight instead of tracked labels.

Fredoka substitutes for the source's proprietary rounded display face — heavy, all caps at hero scale, sentence case everywhere else. Inter carries body copy and never appears bold enough to compete.

03 / Spacing & shape

A 4px grid and two radii that matter.

Eight spacing steps from 2px to 64px. Cards use a generous 28px radius; small sticker badges use 18px; avatars and the logo mark are fully round.

04 / Components

Cards, badges, one link style.

There is no filled button in this system. The primary call to action is an underlined text link; emphasis comes from color blocking and the display face, not chrome.

05 / Do & don't

Where the system holds the line.

Do

  • Keep the canvas a single saturated yellow — there's no neutral-gray background mode here.
  • Reuse the same ink-brown for text and for dark card fills.
  • Keep the primary CTA an underlined text link, not a button.
  • Reserve the float shadow for elements meant to look layered over photography.
  • Set stat numbers in the display face at near-headline weight.

Don't

  • Don't add uppercase tracked mono labels — that's a different system's voice.
  • Don't put a border on cards. Flat color blocking is the separator.
  • Don't mute the accent orange for "professional" contexts.
  • Don't stack more than one shadow level.
  • Don't set body copy in the display face.
